# Keyboard Shortcuts

All keyboard shortcuts are active when the cursor is not in a text input field. If you're typing in the terminal or a search box, keyboard shortcuts are disabled.

## Simulation

| Key | Action |
|-----|--------|
| Space | Pause / Resume simulation |

## Views

| Key | Action |
|-----|--------|
| Tab | Toggle between globe view and topology view |
| V | Top-down view (camera above North Pole) |
| F | Follow the selected node (camera tracks it) |
| Escape | Deselect node / close panel / return to free camera |

## Display Toggles

| Key | Action | Default State |
|-----|--------|---------------|
| L | Toggle ISL links | On |
| G | Toggle ground links | On |
| P | Toggle orbital paths | Off |
| T | Toggle satellite trails | Off |
| H | Toggle historical mode | Off |
| N | Toggle globe rendering mode | Standard |
| ; | Toggle satellite labels | Off |
| ' | Toggle ground station labels | Off |

## Color Modes

| Key | Action |
|-----|--------|
| 1 | Color by routing area |
| 2 | Color by orbital plane |

## Panels

| Key | Action |
|-----|--------|
| ] | Toggle right panel |
| Q | Toggle event filter |
| / | Focus event filter search input |

## Reference Frame

| Key | Action |
|-----|--------|
| I | Toggle inertial / Earth-fixed reference frame |
